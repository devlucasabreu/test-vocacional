## 📋 Sobre o Projeto

Este é um Quiz Interativo de Teste Vocacional desenvolvido puramente com **Vanilla JavaScript**. O projeto foi concebido para guiar usuários na descoberta de perfis profissionais através de uma série de perguntas dinâmicas.

O diferencial técnico deste projeto reside na não utilização de frameworks, focando na implementação de **algoritmos de pontuação personalizados**, um sistema de **gerenciamento de estado manual** para controlar o progresso do usuário e a **manipulação intensiva do DOM** para criar uma interface fluida e reativa. É um excelente exemplo de como conceitos avançados de engenharia de software podem ser aplicados diretamente na linguagem base da web.

## 🚀 Tecnologias Utilizadas

- **HTML5:** Estruturação semântica do formulário e interface do quiz.
- **CSS3:** Estilização moderna, animações de transição entre perguntas e design responsivo.
- **JavaScript (ES6+):** Lógica principal, processamento de dados, controle de fluxo e manipulação dinâmica de elementos sem dependências externas.

## ✨ Funcionalidades

- **Algoritmo de Pontuação:** Lógica que pondera as respostas para classificar o usuário em diferentes categorias profissionais.
- **Gerenciamento de Estado:** Controle preciso do progresso do quiz, permitindo avançar e resetar o estado da aplicação.
- **Interface Dinâmica:** As perguntas e resultados são renderizados em tempo real via JS, evitando recarregamentos de página.
- **Feedback Imediato:** Apresentação clara do perfil vocacional ao final do teste com base nas escolhas realizadas.
